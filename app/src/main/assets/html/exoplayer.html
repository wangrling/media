Developer guide

This guide touches on the pros and cons (利弊) of using ExoPlayer. It shows how to use ExoPlayer to play DASH, SmoothStreaming and HLS adaptive streams, as well as formats such as MP4, M4A, FMP4, WebM, MKV, MP3, Ogg, WAV, MPEG-TS, MPEG-PS, FLV and ADTS(AAC). It also discusses ExoPlayer events, messages, customization and DRM support.

Library overview

An ExoPlayer exposes traditional high-level media player functionality such as the ability to buffer media, play, pause and seek.

Getting started

  1. Create a SimpleExoPlayer instance.
    2. SimpleExoPlayer player = ExoPlayerFactory .newSimpleInstance(context);
  2. Attach the player to a view.
    3. playerView.setPlayer(player);
  3. Prepare the player with a MediaSource to play.
    4. MediaSource videoSource = new ExtractorMediaSource .Factory(dataSourceFactory) .createMediaSource(mp4VideoUri); player.prepare(videoSource);
  4. Controlling the player
    5. PlayerView or PlayerControlView player.addListener(eventListener);
  5. Release the player when done.
    6. ExoPlayer.release();
